About the ElevenSpark demo

The demo covers 11+ maths topics including arithmetic, fractions, decimals, ratios, algebra, measures, area, and data handling. Vocabulary tabs offer synonyms, antonyms, cloze questions, homophones, prefixes, spelling, analogies, and odd-one-out activities.

Each quiz set uses a seeded Mulberry32 generator. This ensures families receive fresh variations every time while tutors can replay the same seed for review.

The static demo keeps state only during your browser session. The upcoming Parent Plan will add secure login, dashboards, and printable packs.

Accessibility

Yes. We design for WCAG 2.2 AA with semantic HTML5, ARIA-labelled components, keyboard support, visible focus, and reduced-motion preferences.

Email access@elevenspark.example with your requirements. We will prioritise compatibility updates.
